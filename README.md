# 🔐 CyberSecurity Threat Detection using Machine Learning & NLP (DistilBERT)

> 🚀 AI-powered system to detect Sensitive APK Usage & Security Vulnerabilities

---

## 📌 Introduction

This project focuses on building a **Cybersecurity Threat Detection System** using **Machine Learning and NLP (Natural Language Processing)**.

It uses **DistilBERT (Transformer Model)** to analyze datasets and identify **potential security threats, vulnerabilities, and sensitive API calls**.

---

## 🎯 Key Features

* 🔍 Sensitive APK Detection
* 🤖 Transformer-based NLP Model (DistilBERT)
* 📊 End-to-End ML Pipeline
* ⚡ Efficient & Scalable
* 🧠 Context-aware threat detection
